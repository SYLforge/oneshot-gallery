# CHADŌ 茶道 — 제작기

## 훅: 멈춤으로서의 제품

다도는 물건이 아니라 갈고 닦은 멈춤이다. CHADŌ는 그 전제를 극단까지
밀어붙인다. 갤러리에서 가장 절제되고, 가장 비어 있고, 가장 느린 엔트리.
유일한 볼거리는 '뺀 것'의 규율이다. 페이지는 하나의 가상 다회(茶會)다.
네 시간의 의식을 한 번의 스크롤로 압축하고, 그 박자는 숨에 맞춘다. 히어로의
글자 茶道는 세로쓰기 다테가키(縦書き)로 한 줄을 기어오르고, 말차빛 구슬이
4-4-4 호흡으로 숨 쉬며, 대나무 차센(茶筅)은 스크롤을 따라 스스로 그려져
한 그릇의 차를 빚어낸다. 그리고 차 그릇이 내려놓아지면, 거기서 끝난다.

이 디자인의 도박은 하나다. 모션이 가득한 갤러리 안에서, 모션을 거의
금당시켜 놓고 — 볼거리가 비트 사이의 *마(間)*에 있는 페이지 — 분명히 자기
자신으로 서는 것이 충분히 경쟁할 수 있다는 것. 절제가 곧 장인정신이다.

## 아트 디렉션

모든 일은 오래된 화지(`#e8e2d4`) 위에서 일어난다. 히어로와 절정에는 한
단계 밝은 `#ece7da`를, 토코노마(床の間) 감실에는 한 단계 어둡고 깊은
`#d9d2c2`를 깐다. 따뜻한 먹(`#2b2620`, 11.61:1)이 말한다 — 결코 순흑이
아니다. 와비사비는 날카로운 검정을 피한다. 단 하나의 강조색은 말차빛
`#5d6b3a`, 페이지 위 살아 있는 유일한 색이다. 작고 값비싼 것들로만 나타난다
— 차센의 대나무 가닥, 차 그릇의 표면, 꽃 하나의 생화(生花), 강조의
헤어라인, 포커스 링.

한 가지 대비 결정이 모든 것을 갈랐다. 브리프가 명시적으로 확인을 요구한
대로, 말차 `#5d6b3a`는 화지 위에서 4.48:1이다 — 본문 AA 기준 4.5에 미치지
못한다. 그래서 말차는 장식적이거나 큰 용도(선, 채움, 헤어라인)로만 쓰고,
본문 크기의 말색 단어는 파생한 `matcha-deep #4a5630`(6.10:1)이 맡는다 —
비트 번호, 그릇의 로마자 이름, 마지막 인장. 브랜드 강조색은 `#5d6b3a`로
두고, 본문용 토큰이 페이지를 정직하게 지킨다. 두 번째 색은 없다. 강조가
필요하면 색이 아니라 공간을 준다.

세리프 둘이 배역을 나눈다. 시뽀리 명조(Shippori Mincho)는 모든 CJK 글리프를
맡는다 — 세로쓰기 서체로서의 수직 메트릭을 먼저 설계했기에, 다테가키 제호와
감실의 두루마리 시를 불만 없이 소화한다. 코모란 가라몬드(Cormorant
Garamond)가 로마자로 응답한다 — 워드마크, 영문 주석, 판권, 호흡 라벨의
로마자 표기. 폰트 스택이 조용히 일한다. 코모란을 앞에, 시뽀리를 뒤에 두면,
마크업 없이 흘러든 한자도 일본 세리프에 도착한다.

## 시그니처 — 다테가키(縦書き)

히어로 제호 茶道는 `writing-mode: vertical-rl; text-orientation: upright`로
세로로 세워진다. 세로쓰기, 일본의 본디 읽기 방향이다. 웹에서 가장 덜 쓰이는
본디의 일본 인터랙션이며, 갤러리의 동아시아 미학 레퍼런스(§4)가 '서구의
일본풍 사이트 거의 쓰지 않는' 티어-1 차별화로 꼽는 것이다. 두 글자는
`aria-label` 뒤에 `aria-hidden` 스팬으로 쪼개져, 제호가 위에서 아래로 —
붓이 지나가는 유일한 방향으로 — 스스로 써 내려간다:

```tsx
<h1 className="chado-tategaki" lang="ja" aria-label="茶道（さどう）">
  {TITLE.map((ch, i) => (
    <span aria-hidden="true" className="chado-tategaki__ch"
      style={{ "--ch-ci": i } as CSSProperties}>{ch}</span>
  ))}
</h1>
```

```css
.chado-tategaki {
  writing-mode: vertical-rl;
  text-orientation: upright;
  font-size: clamp(7rem, 18vw, 14rem);
}
.chado-js .chado-tategaki__ch {
  animation: ch-tategaki-rise 1300ms var(--ch-ease-tea) both;
  animation-delay: calc(180ms + var(--ch-ci, 0) * 220ms);
}
```

다테가키는 의도적으로 귀하게 쓴다. 페이지 전체에 정확히 두 번 — 히어로 제호와
토코노마 두루마리 — 등장해서, 장식이 아니라 신성한 것으로 읽히게 한다.
모션 감소에서는 글리프가 정적으로 렌더링되고(상승 없음), 세로 레이아웃은
그대로다. 쓰기 방향이 곧 콘텐츠이기 때문이다.

## 기법 — 호흡으로 잰 페이지 (scroll-scrub-pinned의 형제)

페이지는 히어로의 호흡 큐로 박자를 맞춘다. 4-4-4 박스 호흡 주기 — 들이쉬기
4초 · 참기 4초 · 내쉬기 4초 · 쉼 2초, 합 14초의 절. 주전자를 마주 앉아 숨
쉬는 이의 박자다. 말차빛 구슬이 호흡을 따라 크기 달라지고, 그 아래 로마자
라벨(`吸う · 止む · 吐く · 間`)이 단계가 바뀌면 교차 사라진다. 페이지 전체가
이에 암묵적으로 박자 맞춰 — 콘텐츠는 도착한 뒤 기다린다. 바쁨의 정반대.

주기는 단계 인덱스로 구동된다. 그래서 스케일(CSS)과 라벨(React)이 결코
어긋나지 않는다:

```ts
// useBreathPace — 프레임당이 아니라 단계당 하나의 setTimeout
const tick = () => {
  phaseIdx = ((phaseIdx + 1) % 4) as BreathPhase;
  setPhase(phaseIdx);
  timeoutId = setTimeout(tick, BREATH_DURATIONS[phaseIdx]);
};
```

스케일 자체는 `data-breath={phase}`에 맞춰 순수 CSS로 돌아간다 —
컴포지터(transform만). 주기는 히어로가 뷰포트를 떠나면(`page.tsx`의
IntersectionObserver가 `cueActive`를 끈다) 그리고 탭이 숨겨지면 멈춘다. 빈
방을 위해 숨 쉴 이유는 없다. 모션 감소에서는 훅이 단계를 '쉼'에 고정하고,
구슬은 그냥 가득 차 고요히 앉아 있다.

이것이 '호흡으로 잰 스크롤'의 정실한 읽이다. 스크롤 속도를 억지로 제한하는
것(휠 하이재킹)은 접근성 위험이며 `prefers-reduced-motion`과 싸운다. 대신
큐는 박자와 넓은 *마*로 — 사용자를 늦추라 안내하는 — 경험적 목표를 이루고,
스크롤의 통제는 온전히 사용자에게 둔다.

## 기법 — 스스로 그려지는 차센 (scroll-scrub-pinned + svg-line-draw)

말차의 준비(点前, 테마에)는 붓질 드로잉이다. 차실(茶室) 장면은 `position:
sticky`로 고정된 채 네 단계의 준비 글이 곁을 흘러가고, 스크롤에 스크럽되는
차센이 대나무 가닥을 한 그릇 속으로 그려 넣는다. 모든 가닥은 `pathLength=1`의
경로이고, 그 대시오프셋은 스크롤 진행도의 역수다:

```css
.chado-chasen__tine {
  stroke-dasharray: 1 1;
  stroke-dashoffset: calc(1 - var(--ch-scrub, 1));
}
/* 차 표면은 준비가 완료될 때만 피어난다:
   스크럽 0.8→1이 투명도 0→1로, 브라우저가 알아서 [0,1] 클램프 */
.chado-bowl__tea {
  fill: var(--ch-matcha);
  opacity: calc(var(--ch-scrub, 1) * 5 - 4);
}
```

고정 자체는 공짜다 — 네이티브 sticky로, 레이아웃은 스크립트하지 않는다:

```css
@media (min-width: 900px) {
  .chado-chashitsu__plate {
    position: sticky;
    top: clamp(4rem, 12vh, 7rem);
    align-self: start;
  }
}
```

스크럽은 작은 rAF 루프다. 매 프레임 정확히 한 번 읽고(장면의 rect) 한 번
쓴다(커스텀 프로퍼티) — 읽기가 쓰기보다 먼저이므로 레이아웃 스래싱이 없다.
원시 진행도는 dt로 정규화한 lerp로 부드럽게 만들어, 120Hz 디스플레이가
60Hz보다 두 배 빠르게 긋는 일이 없도록 했다:

```ts
// useScrollProgress — 한 번 읽고, 한 번 섞고, 한 번 쓴다
const center = rect.top + rect.height / 2;
const span = Math.max(Math.min(rect.height * 0.9, vh * 1.2), 1);
const raw = (vh * 0.62 - center) / span;
const target = Math.min(1, Math.max(0, raw));
value += (target - value) * (1 - Math.pow(0.88, dt / 16.7));
el.style.setProperty("--ch-scrub", value.toFixed(4));
```

IntersectionObserver가 루프를 켜고 끄고, `visibilitychange` 리스너가 탭이
숨겨지면 끈다 — 화면 밖 차실은 프레임 한 개도 쓰지 않는다. 모바일에서
`vh * 1.2` 상한이 중요하다. 단일 열 차실은 길어지는데, 상한이 없으면 그릇이
스크롤 밖으로 사라진 뒤에도 차센이 긋고 있을 것이다.

`prefers-reduced-motion`에서는 훅이 `--ch-scrub: 1`을 한 번 쓰고, 미디어
쿼리가 판을 `position: static`으로 둔다 — 완성된 준비가 정적으로 기다린다.
JavaScript가 없으면 CSS 기본값이 같은 일을 한다. 가닥은 모두 그어져 있고,
그릇은 가득 차 있다.

토코노마의 생화(生花)는 페이지에서 유일한 *시간 기반* 드로잉이다(다른 모든
선은 스크롤 스크럽이거나 정적). 한 줄기와 잎 하나가 리빌에서 한 번, 엇갈리게
그어져 감실이 조립된 채로 도착한다.

## 기법 — 오래된 화지 (feturbulence-texture)

종이는 뷰포트 전체를 채우는 단 하나의 SVG `feTurbulence` 패스다 —
`fractalNoise`, baseFrequency 0.62, 2 옥타브, seed 17 — `feColorMatrix`로
먹색 4.5% 이하 알파에 매핑해 전체에 곱한다. 필름 그레인이 아니라 오래된
종이의 섬유이며, 결코 움직이지 않는다:

```jsx
<filter id="chado-grain-f">
  <feTurbulence type="fractalNoise" baseFrequency="0.62"
    numOctaves={2} seed="17" stitchTiles="stitch" />
  <feColorMatrix type="matrix"
    values="0 0 0 0 0.17  0 0 0 0 0.15  0 0 0 0 0.125  0 0 0 0.045 0" />
</filter>
```

따뜻한 세월 비네시지(가장자리로 `rgba(91,80,58,0.06)`의
`radial-gradient`)가 페이지를 서랍 속에서 고르게 누렇게 뜬 종이로 읽히게
한다. 토코노마는 자기 종이를 더 갖는다 — 한 단계 어두운 `tokonoma` 바닥,
안쪽으로 2.2vw 물러난 1px 먹색 테, 그리고 감실 내부에만 존재하는 두 번째
`feTurbulence`(baseFrequency 0.04)로, 감실이 더 깊고 오래된 종이로 읽힌다.
고요함이 곧 재료다. 어떤 그레인도 움직이지 않는다.

## 비트 사이의 마(間)

랜딩 페이지가 피처를 쌓는 자리에, CHADŌ는 멈춤을 쌓는다. 다도의 네 비트 —
로지(露地), 쓰쿠바이(蹲踞), 와케이세이자쿠(和敬清寂), 주인의 절 — 는
사이에 `clamp(6rem, 18vh, 12rem)`의 공기를 두고 앉는다. 갤러리에서 가장 넓은
요소 간 간격이다. 섹션 여백은 22–26vh. 각 비트는 짧은 이중언어 문장과 그
주변의 거대한 여백일 뿐이다. 섹션은 문장과 문장 사이의 그 하얀 것이 존재하기
위해 존재한다. 이것이 레이아웃으로 실현된 *마(間)* 원리다. 의미있는 멈춤이
곧 디자인이다.

## 접근성 노트

- **모션 감소:** 호흡 구슬이 가득 차 고요히 앉아(링 없음, 라벨 애니메이션
  없음), 다테가키는 정적으로 렌더링되고(상승 없이, 세로 레이아웃은 유지),
  차센 가닥은 모두 그어지며 판은 고정을 푼다(`position: static`), 생화 줄기는
  전환 없이 그어지고, 그릇의 김은 멈추며, 리빌은 움직임 없이 그 자리에.
  하나의 차분하고 온전히 구성된 정적 다도.
- **no-JS:** 모든 사전 리빌 상태는 마운트에서 추가하는 `.chado-js` 뒤에
  숨겨진다. JavaScript가 꺼지면 페이지는 완성된 문서다 — 다도는 종이로
  변환된다. `--ch-scrub` 기본값 1이 차센을 그어진 채, 그릇을 가득 찬 채로
  둔다.
- **의미론:** 다테가키 `h1`은 `茶道（さどう）`를 `aria-label`에 두고 애니메이션
  스팬을 숨긴다. SVG 장면은 이중언어 `<title>`/`<figcaption>`과 함께
  `role="img"`이고, 비트는 순서있는 목록이며, 포커스는 2px
  `matcha-deep`의 `:focus-visible` 링이다.
- **대비:** 먹 11.61:1, 연한 먹 4.58:1, 진한 말차 6.10:1 — 모두 AA 이상.
  4.48:1 말차와 1.4:1 괘선은 결코 글자를 담지 않는다.

## 성능 노트

- 페이지에는 두 개의 타이머가 있다. 호흡 큐의 `setTimeout` 체인(프레임당이
  아니라 단계당 한 번)과 차센 스크럽의 rAF 루프 하나 — 차실이 여유 있는
  옵저버 여백 안에 들어와 있고 탭이 보일 때만 산다. 스크럽 한 프레임은
  `getBoundingClientRect` 한 번과 커스텀 프로퍼티 쓰기 한 번이다.
- 움직이는 것은 모두 `transform`, `opacity`, `stroke-dashoffset`. 고정은
  네이티브 sticky로, 레이아웃을 스크립트하지 않는다.
- 리빌은 하나의 IntersectionObserver에서 `class`와 `transitionDelay`를
  명령형으로 쓴다 — 스크롤 어디에서도 React 리렌더는 없다.
- 이미지도 의존성도 0. 가장 무거운 자산은 두 폰트 패밀리이며, 로마자/CJK
  서브셋은 next/font에 맡긴다.

## 리믹스 아이디어

1. **향 시계.** 차를 향 트레일로 바꾼다. 스크롤에 맞춰 타들어가는 향대;
   호흡 큐는 향도(香道) 손님의 들이쉼이 된다. 다테가키는 제호를 지키고,
   그릇은 향로가 된다.
2. **계절의 키세쓰(季節).** TAWARAYA의 '열두 달' 동작 — 팔레트가 일 년을
   흐른다(벚꽃, 신록, 단풍, 눈) while 의식의 구조는 고정. 계절마다 강조색
   하나, 그릇은 바뀐다.
3. **다도 교실.** 네 비트를 네 개의 클릭 가능한 학습으로 바꿔, 각 고정이
   스크롤이 아니라 요청에 따라 테마에의 다음 단계를 드러낸다 — 차분하고,
   자기 박자로 배우는 다도 학교. 호흡 큐가 각 학습의 읽기 시간을 잰다.
