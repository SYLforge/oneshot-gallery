# 하프라이트 — 제작기

## 훅: 찍히지 않은 영화들의 시사실

어딘가에, 아무도 찍지 않은 영화들의 완성된 악보만 선반 가득 쌓아 둔
작곡가가 있다. 그 영화들에는 슬러그 라인이 있고, 큐 시트가 있고, 엔드
크레딧까지 있다. 다만 존재하지 않을 뿐이다. 이 페이지는 그 스튜디오의
시사실이고, 설정을 문자 그대로 밀어붙인다. 엔트리 어디에도 비디오 파일이
없다. 사진도, 내보낸 프레임 한 장도 없다. 스크럽하는 "푸티지"는 방문자가
도착하는 순간 코드가 그리고, 스코어는 재생을 누르는 순간 합성되며,
크레딧은 "있을 뻔했던 관객들"에게 감사를 전한다.

이 디자인의 도박은 하나다. 부재(不在)도 아트 디렉팅이 된다는 것. 볼 수
없는 영화와 청해야만 들리는 음악이, 자동 재생되는 히어로 비디오보다 더
영화 같을 수 있다는 것.

## 아트 디렉션

여기서 시네마틱 다크는 "다크 모드"가 아니라 영사(映寫)를 뜻한다. 배경
`#0b0b0d`는 램프가 꺼진 스크린이고, 순검정 `#000`은 레터박스 바와 프레임
테두리에만 쓴다. 그래서 화면 영역은 언제나 *빛이 닿은* 면으로 읽힌다.
화면 위 모든 것은 무채색이다 — 목소리는 본 `#e8e4da`, 기계의 메타데이터는
텅스텐 `#a08560` — 예외는 시그널 레드 `#d43425` 하나뿐. 큐 도트로, 포커스
링으로, 삭제된 큐의 등뼈로, 플레이헤드로, 그리고 THE END로만 나타난다.
한 화면에 빨강은 한 번. 빨강은 색상 체계가 아니라 사건이다.

빨강은 가장 어려운 제약이기도 했다. 이 검정 위에서 약 4.0:1 — 본문
텍스트에는 절대 쓸 수 없는 수치다. 그래서 큰 Anton 디스플레이(THE END)와
텍스트가 아닌 표식에만 허락했다. 큐 시트의 "UNUSED" 플래그는 텅스텐이고,
빨강은 그 행의 테두리 등뼈로만 도착한다.

서체 넷은 배역이 넷이다. Anton은 타이틀 카드 — 컨덴스드 대문자, 웨이트
하나. Spectral은 지문(地文)의 목소리, 페이지가 무대 지시로 말할 때는
이탤릭. Space Mono는 모든 기계 표면 — 타임코드, 프레임 카운터, 크레딧
직함. 그리고 Noto Serif KR이 페이지의 모든 한글을 맡는다. 폰트 스택
(`Spectral, Noto Serif KR, serif`; `Space Mono, Noto Serif KR, monospace`)
덕에 한글은 마크업 없이도 라틴 서체를 통과해 제자리에 도착하고,
`:lang(ko)`가 서체를 고정하며 `word-break: keep-all`을 건다.

## 기법 1 — 스프라이트 없는 스프라이트 스크럽

고전적인 스프라이트 스크럽은 수백 장의 JPEG 프레임을 실어 나르며 스크롤
위치에 따라 갈아 끼운다. 하프라이트는 그 문법 — 스크롤 위치가 곧 릴 안의
위치라는 것 — 만 남기고 프레임을 지웠다. 380vh 섹션이 2.39:1 캔버스를
고정하고, `useScrollProgress`가 복도를 `p ∈ [0, 1]`로 사상하며, `p`는
24fps 기준 480개 프레임 번호 중 하나가 된다. 모든 프레임은 *가져오는*
것이 아니라 *계산된다*:

```tsx
// Reel.tsx — 그림이 가진 유일한 시계는 스크롤 진행도다
useScrollProgress(sectionRef, (p) => { targetPRef.current = p; }, !reduced);

// rAF 루프 안: 표시 위치는 필름의 질량처럼 목표를 뒤따른다
shownP += (targetPRef.current - shownP) * SCRUB_LERP * k; // 0.12/프레임
```

시퀀스는 두 개의 숏이다. 숏 A는 "내부. 기억 — 밤." — 기울어진 블라인드
사이 빛의 띠들이 어두운 방을 쓸고 지나간다. 띠 하나하나는 세로 방향
그라디언트고, 알파는 스윕 중심의 가우시안 아래에서 정점을 찍는다. 숏 B는
"외부. 해안 — 황혼." — 창백한 원반이 수평 반짝임 막대들의 바다 위로
떠오르고, 물선에서 잘린다. p 0.45와 0.60 사이에서 두 숏은 디졸브된다 —
smoothstep을 먹인 `globalAlpha`로 겹쳐 그린다:

```tsx
if (p < 0.62) drawShotA(qa);
if (p > 0.45) {
  const x = Math.min(1, (p - 0.45) / 0.15);
  ctx.globalAlpha = x * x * (3 - 2 * x); // smoothstep 디졸브
  drawShotB(Math.min(1, (p - 0.45) / 0.55));
  ctx.globalAlpha = 1;
}
```

릴이 절차적이라는 것은 곧 *결정적*이라는 뜻이기도 하다. 뒤로 스크럽해도
같은 스크래치가 같은 프레임을 가로지른다. 이것이 화면 보호기가 아니라
필름처럼 느껴지게 하는 디테일이다 — 마모 패스는 프레임 번호로 해시되므로,
0312번 프레임은 영원히 0312번 프레임이다:

```tsx
/** 프레임 번호가 같으면 마모도 영원히 같다 — 결정적 해시. */
function hash(n: number, salt: number): number {
  let x = (n * 374761393 + salt * 668265263) | 0;
  x = Math.imul(x ^ (x >>> 13), 1274126177);
  return ((x ^ (x >>> 16)) >>> 0) / 4294967296;
}
```

프레임 위에는 Space Mono HUD가 앉아 있다 — `REEL 02 · 2.39:1`,
`FR 0312`, 24fps 타임코드. 프레임 번호가 바뀔 때만 ref에 명령형으로
써 넣으므로, 카운터는 React 리렌더 없이 카운터답게 째깍인다.

## 기법 2 — 캔버스 파티클: 그레인, 먼지, 마모 패스

그레인은 이 릴의 파티클 시스템이고, 두 개의 시계로 돈다. 160px 타일을
`ImageData`로 한 번만 미리 렌더링한다 — 해시 0.82 위는 밝은 반점, 0.16
아래는 어두운 흠집, 그 사이는 투명 — 그리고
`globalCompositeOperation: "overlay"`, 알파 0.45로 그림 위에 합성한다.
타일의 오프셋은 약 12fps(`GRAIN_MS = 83`)로 다시 뿌려지고, 스크럽 자체는
디스플레이 주사율로 다시 그려진다:

```tsx
if (grainDue) {
  lastGrain = now;
  grainX = Math.random();
  grainY = Math.random();
}
```

60fps의 그레인은 백색소음처럼 보이고, 12fps의 그레인은 필름 스톡처럼
보인다. 시계를 쪼개는 것 — 그림은 디스플레이의 박자로, 그레인은 필름의
박자로 — 이 엔트리에서 가장 값싸고, 영사라는 착각을 가장 확실하게 파는
트릭이다.

그레인 위에서 해시 마모 패스가 진짜 파티클을 뿌린다. 세로 스크래치는
살아 있는 동안 x가 ±1.5px 떨리고(`Math.floor(f / 3)`이 3프레임의 수명을
준다), 프레임당 최대 세 개의 먼지 티끌이 해시된 위치·반지름·극성(밝거나
검거나)으로 태어난다. 프레임마다 해시되는 램프 플리커(3–8% 검정 워시)와
가장자리를 62%까지 잠그는 비네트가 패스를 마무리한다. DOM에도 자체
그레인이 있지만(히어로 위 feTurbulence 타일, 다섯 오프셋 사이를 점프),
파티클이 일하는 곳은 캔버스 쪽이다.

## 기법 3 — 글자 분할 타이틀과 게이트 위브

타이틀 카드는 모순된 두 가지를 원한다. 아홉 글자의 스태거 등장, 그리고
등장이 끝난 뒤에는 하나의 물리적 사물처럼 구는 단어. 그래서 분할은 두
층으로 작동한다. 글리프 하나하나는 자기만의 `animation-delay`를 가진
`aria-hidden` 스팬이고, 단어 전체는 단 하나의 "위브" 래퍼에 앉아 있으며,
접근 가능한 이름은 `h1`이 갖는다:

```tsx
<h1 className="halflight-title" id="halflight-title" aria-label={TITLE}>
  <span className="halflight-title__weave" aria-hidden="true">
    {TITLE.split("").map((ch, i) => (
      <span key={`${ch}-${i}`} className="halflight-title__ch"
        style={{ animationDelay: `${220 + i * 55}ms` }}>
        {ch}
      </span>
    ))}
  </span>
</h1>
```

스크린 리더는 "HALFLIGHT"를 한 번에 읽는다. 눈으로 보는 방문자는 글자가
레터박스에서 하나씩 떠오르는 것을 본다(`translateY(0.35em) → 0`, 700ms,
55ms 스태거). 그다음은 래퍼의 차례다. 게이트 위브는 아홉 개의 서브픽셀
위치(±0.7px x, ±0.9px y)를 도는 7.3s 키프레임 루프인데, 타이밍이
`steps(1, end)`다 — 단어는 위치 사이를 *점프*하고 멈춘다. 헐거운 영사기
게이트 안의 필름이 하는 짓이 정확히 그것이다. 같은 키프레임에 이징을
먹이면 타이틀은 젤리가 된다. `steps(1)`, 아니면 아예 하지 않기.

두 층 모두 `.halflight-js` 뒤에 게이트되어 있고(JS가 없으면 숨겨진
텍스트도 없다 — 사전 상태는 `animation-fill-mode: both`가 나른다), 모션
최소화 미디어 쿼리에서 둘 다 죽는다.

## 스코어 — 3초 늦게, 의도적으로

오디오는 meta.json의 태그가 아니라 이 엔트리의 두 번째 절차적 매체다.
PLAY를 누르면 — 그리고 오직 그때만 — Web Audio 그래프가 지어진다. 룸
톤(55Hz 사인과 일부러 살짝 높인 110.7Hz, 그 위 12도의 트라이앵글, 전부
260Hz 로패스 통과)이 1.6초에 걸쳐 차오른다. 현악 — 디튠된 톱니파 둘과
빈 5도, 숨 쉬는 로패스 통과 — 은 페이드가 아니라 *예약*이다.
`osc.start(now + 3)`. 큐 시트의 "현악은 3초 늦게, 의도적으로 들어온다"는
문장은 실행 가능한 코드다:

```ts
// useAudioEngine.ts — 카피를 문자 그대로
const enter = now + STRINGS_DELAY_S;
strEnv.gain.setValueAtTime(0.0001, enter);
strEnv.gain.linearRampToValueAtTime(0.17, enter + STRINGS_SWELL_S);
voice("sawtooth", 220, 0.5, strFilter, enter);
```

0.2Hz 미만의 LFO 둘(필터 주파수 ±320Hz, 스웰 ±0.24)이 드론이 멈춰 있거나
루프처럼 들리는 것을 막는다. 소리가 나는 동안 AnalyserNode가 캔버스의
스코어 라인을 먹인다. 왼쪽으로 행진하는 RMS 엔벨로프, 그 뒤에 텅스텐으로
어른거리는 원시 오실로스코프, 붉은 플레이헤드, 그리고 18% 지점의 고정된
붉은 표시 — 현악의 입장이다. 모션 최소화의 정적인 그림에서도 그 표시는
남는다.

모든 출구는 해체다. 정지는 마스터를 0.55초에 걸쳐 내리고, 모든
오실레이터를 세우고, 그래프를 끊고, AudioContext를 닫는다. 섹션을 떠날
때(IntersectionObserver), 언마운트, `pagehide` — 전부 같은 함수에
도착한다. 노드가 새거나 화면 밖에서 소리가 이어지는 경로는 없다.

## 모션의 수학

- **스크럽 lerp 0.12/프레임.** 반감기 ≈ `ln(0.5)/ln(1 − 0.12)` ≈ 5.4프레임
  ≈ 60fps에서 90ms(`dt / 16.7`로 정규화). 릴에 질량이 생길 만큼의 지연 —
  휠을 튕기면 한 박자 늦게 도착한다, 실제 필름이 여유분을 감아 들이듯 —
  이지만 프레임 카운터가 스크롤바와 따로 노는 느낌은 없는 정도.
- **디졸브 구간(0.45–0.60)** 은 복도의 15% ≈ 스크롤 57vh. 디졸브 한가운데
  멈춰 이중 이미지를 들여다볼 수 있을 만큼 길고, 어느 숏도 서로에게
  오염되지 않을 만큼 짧다.
- **그레인 83ms**, 위브 7.3s, 블링크 8.1s — 세 주기는 공약수가 없어서
  어둠이 눈에 띄게 반복되는 일이 없다.
- **오디오의 비대칭, 인 1.6s / 아웃 0.55s.** 드론은 방(房)처럼 도착하고
  스위치처럼 떠나야 한다 — 릴리스가 길면 방문자가 다음으로 스크롤한 것
  위에 소리가 번진다.

## 성능 노트

- 릴의 rAF 루프는 하나. 스크럽이 움직이지 않았고(> 0.0004), 프레임 번호가
  그대로고, 83ms 그레인 시계도 울리지 않았으면 그리지 않고 일찍 돌아간다 —
  가만히 고정된 릴의 비용은 리페인트가 아니라 비교 연산 하나다.
- 루프는 화면 밖(IntersectionObserver, 여유 120px)과 숨긴
  탭(`visibilitychange`)에서 완전히 멈춘다. `dt`는 48ms로 클램프되어
  백그라운드 탭이 덜컹거림을 만들지 못한다.
- DPR은 2로 캡. 그레인 타일은 프레임당 약 21회, 미리 렌더링된 캔버스에서
  그려진다 — 초기화 이후 픽셀 단위 작업은 없다. 숏의 그라디언트는
  프레임마다 다시 만들어지지만(약 25개), 2.39:1 크기에서 프레임 예산에
  넉넉히 들어온다.
- 파형 루프는 `playing && !reduced`인 동안에만 존재하고, 정지는 같은
  이펙트 클린업에서 루프를 취소한다.
- HUD 카운터, 큐 마크 불투명도, 크레딧 롤의 `--hl-roll-p`는 전부 ref와
  CSS 변수에 대한 명령형 쓰기다 — 스크롤 프레임당 React 리렌더는 0회.
  페이지의 상태 변화는 재생/정지와 한 번뿐인 `.halflight-js` 클래스가
  전부다.
- DOM 애니메이션은 transform/opacity뿐. 스크럽 복도는 `.halflight-js`
  아래에서만 존재하므로, JS 없는 페이지는 죽은 스크롤 없이 보통 높이다.

## 접근성 노트

- **모션 최소화:** `usePrefersReducedMotion`(`matchMedia` 위의
  `useSyncExternalStore`)이 릴을 잘 구성된 정지 프레임 하나(p = 0.78,
  디졸브의 끝자락)에 세우고, 스크롤 사상을 끄고, 두 스크럽 복도를 보통
  흐름으로 접는다. 미디어 쿼리는 위브·글자 상승·그레인·블링크·리빌을
  죽인다. 스코어 버튼은 여전히 작동한다 — 모션 최소화는 소리 최소화가
  아니다 — 다만 파형은 정적인 그림으로 그려진다.
- **오디오 동의:** AudioContext는 클릭 핸들러 안에서 태어난다. 자동
  재생 경로는 없고, 버튼은 `aria-pressed`와 눈에 보이는 PLAY/STOP 상태를
  가진 네이티브 `<button>`이다.
- **JS 없음:** 숨김 사전 상태(`[data-reveal]`, 복도, 플레이어 컨트롤)는
  전부 `.halflight-js` 뒤에 있다. 자바스크립트가 없어도 페이지는 완결된,
  읽을 수 있는 문서다. 릴 프레임에는 CSS 정지 화면이 있다.
- **시맨틱:** 두 캔버스 모두 완전한 이중 언어 설명이 붙은 `role="img"`
  래퍼 안에 있고 캔버스 요소 자체는 `aria-hidden`이다. HUD와 큐 도트는
  장식이라 트리에서 숨겼다. 삭제된 큐는 색만으로 말하지 않는다 — 진짜
  `<s>` 요소에 UNUSED · 미사용 플래그를 함께 단다.
- **대비:** 본 15.5:1, 본-딤 6.6:1, 텅스텐 5.6:1 — 스크린 블랙 위에서
  전부 AA 이상. 빨강(4.0:1)은 큰 디스플레이 타입과 비텍스트 표식에
  가두었고, HUD는 릴의 가장 밝은 프레임 위에서도 읽히도록 스크림 위에
  앉아 있다.

## 리믹스 아이디어

1. **라디오 드라마.** 엔진은 그대로, 그림만 지운다. 릴 캔버스는 천천히
   패닝하는 VU 미터 풍경이 되고, 큐 시트는 방송 일지가 되고, Web Audio
   패치에는 밴드패스 "튜닝" LFO가 걸린 노이즈 베드(버퍼 소스)가 더해진다.
   스크럽은 잡음 사이를 오간다.
2. **없는 장소의 사진가.** 두 숏을 네 숏으로 바꾸고 밀착 인화지 모드로:
   복도가 프레임 스트립을 가로질러 스크럽되고, 멈춰 있으면 현재 프레임이
   "현상"된다(그레인이 가라앉고 비네트가 깊어진다). 빨강은 선택된 프레임에
   그리는 그리스 펜슬 동그라미가 된다.
3. **플라네타륨.** 같은 결정적 해시로 마모 대신 별밭을 그린다. 스크롤이
   하룻밤의 하늘을 스크럽하고, 드론은 시각에 따라 디튠되는 패드가 되고,
   크레딧 롤에는 끝내 이름 붙지 못한 별자리들이 올라간다.
